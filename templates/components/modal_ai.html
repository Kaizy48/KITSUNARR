Editor de Prompt Base

Guía rápida

Un prompt custom reemplaza por completo el prompt maestro de Kitsunarr. Úsalo solo si quieres ajustar reglas, idioma o estilo de normalización.

Contrato obligatorio

La respuesta del modelo debe ser JSON puro y válido, sin Markdown ni explicaciones:

{
  "translated_title": "Título final",
  "tvdb_id": "ID o null"
}
Variables disponibles
  • {title}: título original del tracker.
  • {description}: sinopsis extraída del tracker.
  • {tvdb_candidates}: candidatos TVDB en JSON.
Buenas prácticas
  • Conserva intacto el primer bloque de fansub.
  • Usa el nombre exacto del candidato TVDB cuando haya match.
  • Incluye siempre temporada: S01 o pack S01-S04.
  • No uses IDs fuera de {tvdb_candidates}.
  • No bases el match en codec, audio, subs o resolución.

Tip: dejar el campo vacío restaura el prompt maestro interno, que también aplica validaciones de seguridad después de la IA.
