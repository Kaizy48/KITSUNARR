Seguridad y API (Kitsunarr)

Esta clave es necesaria para que Sonarr se pueda conectar a Kitsunarr de forma segura.

TheTVDB

La información oficial de las series y los pósters es proporcionada por TheTVDB.com. Regístrate como desarrollador para obtener una clave gratuita.

Búsqueda Automática TVDB
Busca metadatos y IDs en segundo plano (Requiere Clave API guardada).

Funciones Avanzadas de Procesamiento

Controla el comportamiento general de la Inteligencia Artificial y la automatización en segundo plano.

Motor General de IA
Habilita el uso de IA para normalizar títulos y cruzar metadatos con TVDB.
Procesamiento Automático
El sistema buscará torrents pendientes cada 60s y los procesará en segundo plano.

Conexión del Motor LLM

Configura las credenciales y el modelo del proveedor de inteligencia artificial.

Límites de Cuota (API)

Requests Per Minute.
Define la velocidad de escaneo. Kitsunarr calculará pausas automáticas para no exceder este número de peticiones por minuto.

Tokens Per Minute.
Volumen máximo de "palabras" que la API procesa por minuto. (Normalmente 250k a 1 Millón en planes gratuitos).

Requests Per Day.
Tope diario. El trabajador automático se pausará hasta el día siguiente si alcanza este número de torrents procesados.

Estos límites protegen tu cuenta de bloqueos temporales (Errores 429) durante el procesamiento en segundo plano.
💡 Consejo: Si utilizas esta misma API Key para otras aplicaciones o bots, es recomendable configurar aquí valores menores al máximo permitido por tu proveedor. Así reservarás cupo y Kitsunarr no agotará tus recursos diarios.
